# Fuentes oficiales para catálogos masivos (México)

## Educación (escuelas, bachilleratos, universidades)
1. SEP / datos.gob.mx - Catálogo de Centros de Trabajo SEP (CSV, por año y por estado)
- https://www.datos.gob.mx/dataset/catalogo_centros_trabajo_sep

2. SIGED - Datos abiertos y Catálogo de Centros de Trabajo
- https://sigedqa.sep.gob.mx/SIGED_2.0/datos_abiertos.html

3. ANUIES - Directorio de Instituciones de Educación Superior (para depurar IES)
- https://www.anuies.mx/anuies/instituciones-de-educacion-superior/

## Salud (hospitales, clínicas y unidades médicas)
1. DGIS / Secretaría de Salud - Catálogo CLUES (XLSX)
- https://www.dgis.salud.gob.mx/contenidos/intercambio/clues_gobmx.html

2. Subsistema CLUES (metodología y consulta)
- https://www.dgis.salud.gob.mx/contenidos/sinais/s_clues.html

## Complemento económico/geográfico (validación de establecimientos)
1. INEGI - DENUE (consulta y descarga masiva)
- https://www.inegi.org.mx/app/mapa/denue/
- https://www.inegi.org.mx/app/descarga/

## Recomendación técnica
- Base primaria educación: SEP (CCT).
- Base primaria salud: CLUES.
- Base de apoyo para normalización/altas no cubiertas: DENUE (INEGI).
- Para universidades de alto nivel y nombres institucionales: ANUIES.

## Nota sobre carreras/especialidades
- INEGI no es la mejor fuente primaria para catálogo de carreras académicas.
- Para especialidades médicas conviene mantener catálogo maestro interno + validación con referencias académicas oficiales (PUEM/UNAM).
